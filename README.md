## Installation

> CMD: In Projektordner wechseln

    cd

> CMD: git clone & git remote remove orgin

    git clone https://github.com/tBrause/npm-vite-react-sass.git .; git remote remove origin

> CMD: Fast start

    npm install; npm run build; npm run dev
