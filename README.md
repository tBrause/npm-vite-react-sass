# Installation DEV-Entwicklungsumgebung

> In Projektordner wechseln

    cd

> git clone

    git clone https://github.com/tBrause/npm-vite-react-sass.git .; git remote remove origin

> CMD: npm install

    npm install

> CMD: npm run build

    npm run build

> CMD: npm run dev

    npm run dev

> CMD: npm run preview

    npm run preview
